## Installation

### Prerequisites

Ensure you have the following installed on your system:

- **C++ compiler (e.g., g++)**
- **Java Development Kit (JDK)**
- **CMake** (for building the C++ part of the project)
- **Gradle or Maven** (for managing Java dependencies)
